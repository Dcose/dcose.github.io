### BFC（Block Formatting Context）

#### 是什么
实际就是页面一个独立的容器，里面的子元素不影响外面的元素

可以包含内部浮动元素
可以排除外部浮动带来的影响
阻止外边距重叠
解决 `margin` 塌陷
