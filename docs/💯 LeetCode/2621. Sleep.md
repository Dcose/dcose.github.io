# 2621. Sleep (睡眠函数)

> [题目 :airplane::airplane:](https://leetcode.cn/problems/counter/description/)
>
> 难度：**简单**
>
> 标签：SetTimeout

### 解一

```javascript
/**
 * @param {number} millis
 */
async function sleep(millis) {
	await new Promise((s) => {
		setTimeout(s, millis);
	});
}

/**
 * let t = Date.now()
 * sleep(100).then(() => console.log(Date.now() - t)) // 100
 */
```
